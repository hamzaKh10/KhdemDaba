# KhdemDaba

A simple **website prototype** that presents the idea of KhdemDaba: helping people in Morocco find work faster (students, beginners, freelancers, and practical jobs like coffee, delivery, cleaning, electrician, handyman…).

- Tech: **HTML + CSS + JavaScript** (no backend)
- Data: **prototype mode** (saved locally on your device)
- Languages: **EN / العربية / الدارجة**
